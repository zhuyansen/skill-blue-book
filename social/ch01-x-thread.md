### 第 5 条（为什么爆发在 2026）· ~130 字

```
Skill 这个想法在 2024 年就有了。
为什么到 2026 年 Q1 才真正爆发？

因为之前装一个能力扩展要这 6 步：
1 下源码 → 2 配环境 → 3 启进程
4 配端口 → 5 重启 IDE → 6 运气好工作

Skill 变成了 1 步：
  扔进 ~/.claude/skills/

门槛低 6 倍。用户基数大 10 倍。
2026-03 单月新增 27,720 个 skill。
```

**配图建议**：用 `ch03-fig2-supply-surge.png`（月度供给柱状图，红色 3 月爆炸）

### 第 6 条（对比其他扩展）· ~140 字

```
有人问：System Prompt / RAG / Tool / MCP / Skill —
这五个凭啥 Skill 赢？

我这么理解：

• Tool 是"四肢"（能做什么）
• MCP 是"独立的手"（跨进程能做什么）
• RAG 是"百科全书"（事实检索）
• System Prompt 是"永久人格"（一直在线）
• Skill 是"教练笔记"（该怎么做）

Agent 真正最稀缺的，是**教练笔记**。
```

**配图**：`social/ch01-fig3-five-extensions.png`（5 种扩展机制对比，SKILL 橙色高亮，1:1 方图）
